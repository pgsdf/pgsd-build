PGSD Desktop Environment
========================

This directory contains PGSD-specific resources and documentation.

Directory Structure:
  - images/          System images for installation (populated during ISO build)
  - themes/          Desktop themes and visual customizations
  - scripts/         Utility scripts for system management
  - docs/            User documentation

Getting Started:
  1. Arcan/Durden is configured to auto-start on tty1
  2. Configuration files are in ~/.arcan/durden/
  3. Use Meta+Enter to open a terminal (foot)
  4. Use Meta+D to open the launcher

Useful Commands:
  - beadm list                  List boot environments
  - beadm create <name>         Create new boot environment
  - pkg update && pkg upgrade   Update packages
  - zfs list                    List ZFS datasets
