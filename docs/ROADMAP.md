# PGSD Roadmap (Prototype Snapshot)

Short‑term goals:

- Solidify the `pgsdbuild` CLI and internal structure
- Define a working `pgsd-desktop` image recipe
- Define a working `pgsd-bootenv-arcan` variant
- Get a minimal Arcan/Durden boot ISO that can launch `pgsd-inst`
- Implement the ZFS‑based installer pipeline end‑to‑end

Longer‑term goals:

- Multiple image profiles (desktop, minimal, server, OEM)
- sysrebase‑style patch updates
- aarch64 support
- Secure Boot integration
